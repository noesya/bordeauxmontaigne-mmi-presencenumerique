Si vous vous dirigez vers un parcours dans le développement web ou la programmation, github sera votre plus fidèle compagnon. Hébergement, contributions, documentation, partage et communauté sont les termes qui régissent sur la plateforme.

Github peut vous servir de portfolio, vous pouvez recenser vos projets, vos contributions à des projets open source. La plateforme permet également de travailler de manière collaborative, un point non négligeable qui vous servira lorsque vous serez amené à travailler en équipe au sein d’une entreprise.

Au-delà du fait que cet outil vous sera essentiel en tant que développeur, il vous servira également sur le plan professionnel. N’importe quel recruteur cherchant un développeur voudra systématiquement pouvoir consulter son profil github. Cela lui permet de voir vos repositories (publics), les contributions, une vue d'ensemble via votre calendrier. En effet, cela en dit long sur le développeur que vous êtes. Qu’on ne se méprenne pas, ne pas avoir un profil attirant sur github ne fera pas de vous un mauvais candidat, mais l’inverse vous garantie de ressortir du lot.
